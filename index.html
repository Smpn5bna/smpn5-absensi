📋 Panduan Lengkap Setup GitHub (10 Menit):

🔸 STEP 1: Buat Akun GitHub (Jika Belum Punya)

• Kunjungi GitHub.com

• Klik "Sign up" dan buat akun gratis

• Verifikasi email Anda

• Login ke akun GitHub

🔸 STEP 2: Buat Repository Baru

• Klik tombol hijau "New" atau kunjungi github.com/new

• Repository name: smpn5-absensi

• Description: Sistem Absensi Keterlambatan Siswa

• Pilih Public (untuk GitHub Pages gratis)

• Centang "Add a README file"

• Klik "Create repository"

🔸 STEP 3: Buat Personal Access Token

• Klik foto profil (kanan atas) → Settings

• Scroll ke bawah → Developer settings

• Personal access tokens → Tokens (classic)

• Klik "Generate new token" → "Generate new token (classic)"

• Note: SMPN5 Absensi Token

• Expiration: No expiration (atau 1 year)

• Centang scope: ✅ repo (semua sub-checkbox)

• Klik "Generate token"

• PENTING: Copy token dan simpan! (dimulai dengan ghp_)

🔸 STEP 4: Upload File HTML ke Repository

• Di repository, klik "uploading an existing file"

• Upload file HTML aplikasi ini

• Rename file menjadi index.html

• Scroll ke bawah, isi commit message: "Upload aplikasi absensi"

• Klik "Commit changes"

🔸 STEP 5: Edit Konfigurasi di File HTML

• Klik file "index.html" di repository

• Klik tombol pensil (✏️) untuk edit

• Cari baris 325-330 (GITHUB_CONFIG)

• Ganti dengan data Anda:

username: 'username-github-anda',
repository: 'smpn5-absensi',
token: 'ghp_token-yang-anda-copy',

• Scroll ke bawah, klik "Commit changes"

🔸 STEP 6: Aktifkan GitHub Pages

• Di repository, klik tab "Settings"

• Scroll ke bawah cari "Pages" (sidebar kiri)

• Source: Deploy from a branch

• Branch: main

• Folder: / (root)

• Klik "Save"

• Tunggu 2-3 menit, aplikasi akan tersedia di:

https://username-anda.github.io/smpn5-absensi

🔸 STEP 7: Test & Verifikasi

• Buka aplikasi di URL GitHub Pages

• Klik tombol "🧪 Test Koneksi" di bawah

• Jika berhasil: Status akan berubah "🌐 GitHub Connected"

• Coba input data siswa dan absensi

• Data akan otomatis tersimpan ke GitHub!

⚡ Config GitHub di Kode:

const GITHUB_CONFIG = {
  username: 'your-github-username',
  repository: 'smpn5-absensi',
  token: 'ghp_your-token-here',
  branch: 'main'
};

• Pilih kelas, lalu paste semua nama sekaligus

📋 Format File Excel yang Diperlukan:

• Kolom A: Kelas (contoh: 7-1, 7-2, 8-1, dst)

• Kolom B: Nama Siswa (contoh: Ahmad Rizki Maulana)

• File format: .csv atau .xlsx

• Tanpa header - langsung data mulai baris 1
